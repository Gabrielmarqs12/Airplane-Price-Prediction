# Airplane Price Prediction

This is a model trained to predict aircraft values based on their characteristics using linear regression and libraries such as seaborn, pandas and scikit-learn.

Dataset from Kaggle: **[Airplane Price Prediction](https://www.kaggle.com/datasets/rafsunahmad/plane-price-prediction)**

Results may be seen at **[- W I P -](https://github.com/Gabrielmarqs12/Airplane-Price-Prediction)**
